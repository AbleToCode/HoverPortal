# HoverPortal 打包发布文档

## 发布版本

| 版本类型 | 文件大小 | 说明 | 位置 |
|---------|---------|------|------|
| **FDD** (框架依赖) | ~870 KB | 需用户安装 .NET 7 Runtime | `publish/fdd/` |
| **SCD** (自包含) | ~68.6 MB | 独立运行，无需安装 Runtime | `publish/scd/` |

---

## 快速打包命令

### 框架依赖版本 (FDD)
```powershell
dotnet publish src/HoverPortal/HoverPortal.csproj -c Release -r win-x64 --self-contained false -o publish/fdd
```

**优点**: 体积极小 (~870KB)  
**缺点**: 用户需预先安装 [.NET 7 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/7.0)

---

### 自包含版本 (SCD)
```powershell
dotnet publish src/HoverPortal/HoverPortal.csproj -c Release -r win-x64 --self-contained true -o publish/scd
```

**优点**: 开箱即用，无任何依赖  
**缺点**: 体积较大 (~68MB)

---

## 分发建议

| 场景 | 推荐版本 |
|------|---------|
| 开发者/技术用户 | FDD - 体积小，下载快 |
| 普通终端用户 | SCD - 无需额外安装 |
| 企业内部部署 | FDD + 统一部署 Runtime |

---

## 当前发布目录结构

```
publish/
├── fdd/                    # 框架依赖版本
│   └── HoverPortal.exe     # ~870 KB
└── scd/                    # 自包含版本
    └── HoverPortal.exe     # ~68.6 MB
```

---

## 打包为 ZIP 分发

```powershell
# FDD 版本
Compress-Archive -Path publish\fdd\* -DestinationPath HoverPortal_v1.2_fdd.zip

# SCD 版本  
Compress-Archive -Path publish\scd\* -DestinationPath HoverPortal_v1.2_scd.zip
```
